DESCRIPTION
===========

This cookbook installs ack-grep, plain and simple.

REQUIREMENTS
============

Requires that the platform has a package named ack-grep or ack.

USAGE
=====

Simply add the "recipe[ack-grep]" to your run\_list or role and you'd be
good to go!
